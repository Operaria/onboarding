# Link para enviar — Levantamiento Web / Studio

    https://onboarding.operaria.cl/web/<cliente>?negocio=<negocio>

Ejemplo:
    https://onboarding.operaria.cl/web/pedro?negocio=tienda-luna
    → muestra "Pedro" / "Tienda Luna"

Regla: minúsculas, palabras con guiones, sin tildes ni espacios.
Al enviar, llega un PDF al correo.
